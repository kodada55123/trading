跨裝置同步設定

GitHub Gist 同步

筆記將儲存在你的 GitHub Gist，電腦與手機自動同步。只需設定一次。

前往 github.com/settings/tokens，勾選 gist 權限，建立 Token 並複製。
前往 gist.github.com，建立新 Secret Gist：檔名填 notes.json，內容填 {}，點「Create secret gist」。
複製 Gist 頁面網址中的 ID（網址最後那段 32 位英數字），填入下方。
